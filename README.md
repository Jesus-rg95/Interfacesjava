# Calculo de Area y Perimetro de Figuras Geometricas

Este ejercicio muestra  el uso de interfaces para el cualculo del área y perimetro de algunas figuras geometricas.

## Figuras Utilizadas 

-Triangulo
-Cuadrado
-Rectángulo
-Rombo
-Rombiode
-Trapecio
